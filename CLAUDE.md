## Project Overview

This is a small demo project for displaying public holidays using Vaadin and Spring Boot. The repository contains both the frontend UI components and backend services to fetch and display holiday data.

## Technologies Used

- Vaadin 25
- Spring Boot 4

## External APIs

- [Nager.Date Public Holiday API](https://date.nager.at) - Used to fetch public holiday data.
  - Documentation in ./holiday-api.md
